## Why this example

Single-skill flow showing Tanker's static site craft:

- `/ui-hunt` → reference brief.
- `/design-shotgun` → 4 working HTML prototypes, owner picks one.
- `/static-site-replicator` → final polished single-file `index.html`.
- `site-eval` agent → 9-dimension audit before delivery.

Ends with a deployable single-file site that doesn't read like AI slop.

## Expected runtime

- ~10 minutes.
- Cost: ~$1–2.
- No deploy unless you append `/deploy`.
